# 🍕 Pizza Sales Analysis SQL Project

📊 Overview
This project analyzes pizza sales data using SQL to extract business insights.

🎯 Objectives
- Analyze total revenue and orders
- Identify top-selling pizzas
- Understand customer preferences
- Perform category-wise analysis

🛠️ Tools Used
- MySQL
- SQL (Joins, Subqueries, Window Functions)

📌 Key Insights
- Top 5 pizzas drive major sales
- Certain categories contribute highest revenue
- Customer preference varies by size

📂 Files
- Queries.sql → All SQL queries used in analysis
- tables.sql →  SQL queries used in table creation
